# PPT 故事线

为**汇报类**演示设计叙事结构与分页大纲（标题 + 要点）。默认输出 Markdown 大纲；用户明确要求时再生成 pptx。

## 何时启用

- 做汇报 PPT、路演、项目复盘、投标讲解结构
- **不是**单纯美化已有 slides，而是 **先定故事线**

## 执行步骤

1. **澄清**：受众、场景、时长、页数上限、核心结论一句话
2. 设计故事线：**背景 → 问题/机会 → 方案 → 证据 → 收益 → 行动呼吁**
3. 输出 **分页大纲**（每页：标题 + 3 条以内要点 + 演讲者备注可选）
4. 用户确认后，若需要文件：调用 `office_ppt_create`，把每页标题与要点传入

## 推荐内置工具

- `office_ppt_create` — 由大纲生成 `.pptx`（需用户确认内容后再调用）

## 输出模板

```markdown
# 演示大纲：[主题]

## 核心信息（电梯演讲）
一句话 …

## 受众与场景
…

## 故事线
1. 背景 …
2. …

## 分页（共 N 页）
### 第1页｜标题（动词开头）
- 要点
- …

### 第2页｜…
…

## 附录建议（可选）
- 备用数据页 …
```

## 写作要求

- 标题 **动词或结论式**，避免「概述」「其他」
- 默认 8～12 页；用户指定页数时严格遵守
- 无数据时向用户要素材，不编造业绩数字

## 禁止

- 未确认就生成 pptx
- 把 Markdown 大纲贴完当作「已做好 PPT 文件」
